## How to Run:
1. Install packages
```shell
npm install && npm install -g webpack-dashboard
```

2. To run the tests & webpack-dev-server run:
```shell
npm start
```

3. To run the dev-server using webpack-dashboard run:
```shell
npm run dev
```
